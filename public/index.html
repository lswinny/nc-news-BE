NC News API Documentation

Hello World, let's navigate NC News!

Navigation

/api/health

  • GET: Check that the server is running properly

/api/articles

  • GET: Retrieve a list of all articles

This endpoint supports the following query parameters:

  1. sort_by: Sort articles by title, topic, author, votes, or created_at.
    Example: /api/articles?sort_by=title&order=asc
  2. topic: Filter articles by topic.
    Example: /api/articles?topic=cats

/api/articles/:article_id

  • GET: Retrieve a specific article by its ID
  • PATCH: Update fields of a specific article

/api/articles/:article_id/comments

  • GET: Get all comments for a specific article
  • POST: Add a new comment to a specific article

/api/comments/:comment_id

  • DELETE: Remove a specific comment by its ID

/api/topics

  • GET: Retrieve a list of all topics

/api/users

  • GET: Retrieve a list of all users
